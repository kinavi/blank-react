# About

**blank-react** is a blank for creating projects.

Responsible for the assembly **Webpack 4.43**

##### The form is designed to work on:
 
- [x] React
- [x] Redux
- [x] React router
- [x] TS
- [x] Sass
- [x] Eslint
- [x] Jest
- [ ] Express
- [ ] MongoDB

### Main scripts

- **yarn dev** - starting a local webpack server 
- **yarn build:client** - starts front end assembly
